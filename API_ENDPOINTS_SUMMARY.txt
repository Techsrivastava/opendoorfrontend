1. User enters phone number
2. Call: POST /api/customers/send-otp
3. OTP sent to phone via Message Central
4. User enters OTP
5. Call: POST /api/customers/verify-otp
6. Account created + JWT token returned
7. Save token in localStorage
8. Redirect to profile completion

LOGIN:
1. User enters phone number
2. Call: POST /api/customers/login/send-otp
3. OTP sent to phone
4. User enters OTP
5. Call: POST /api/customers/login/verify-otp
6. JWT token returned
7. Save token in localStorage
8. Redirect to dashboard
